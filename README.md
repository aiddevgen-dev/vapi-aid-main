# VAPI Aid - AI-Powered Call Center

VAPI Aid is an advanced AI-powered call center solution that provides real-time transcription, customer insights, and intelligent agent assistance to enhance customer service operations.

## Features

- **Real-time Transcription**: Live speech-to-text using AssemblyAI
- **AI-Powered Suggestions**: Intelligent response recommendations for agents
- **Customer Profile Management**: Comprehensive customer information and history
- **Call Analytics**: Detailed call metrics and performance insights
- **Multi-channel Support**: Voice calls with integrated chat capabilities
- **Agent Dashboard**: Unified interface for managing customer interactions

## Technology Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Supabase (PostgreSQL, Auth, Real-time)
- **Voice Processing**: Twilio Voice SDK
- **Speech Recognition**: AssemblyAI
- **AI Integration**: OpenAI GPT for intelligent suggestions

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- Supabase account
- Twilio account
- AssemblyAI account

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd vapi-aid
```

2. Install dependencies:
```bash
npm install
```

3. Set up environment variables:
Create a `.env` file with your API keys and configuration.

4. Start the development server:
```bash
npm run dev
```

## Configuration

1. **Supabase Setup**: Configure your Supabase project with the required tables and policies
2. **Twilio Integration**: Set up Twilio Voice webhooks and credentials
3. **AssemblyAI**: Configure real-time transcription settings
4. **AI Models**: Set up OpenAI integration for intelligent suggestions

## Deployment

The application can be deployed to any modern hosting platform that supports Node.js applications.
